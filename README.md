## Iniciar Servicio

Primero completar las variables de conexion a la base de datos que se encuentran en el archivo de variables .env

luego:

```bash
npm install

npm run start

```

Abrir [http://localhost:2000](http://localhost:2000) en tu navegador para ver el resultado.
